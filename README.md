# Desafio 

## Requisitos

- [Node.js](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com)

### Dependências

- [React.js](https://reactjs.org/)
- [Nodemon](https://nodemon.io)
- [Express](https://expressjs.com)

### Instalação

- backend 

   `yarn install`
   
- frontend 

   `yarn install`
   
### Execução

- backend 

   `yarn dev`
   
- frontend 

   `yarn start`
